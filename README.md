## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm / yarn / pnpm

---

### Installation

```bash
git clone https://github.com/prodigy4801/personal_dev_portfolio.git
cd personal_dev_portfolio
npm install
```

---

### Development

```bash
npm run dev
```

App runs on:

```
http://localhost:5173
```

---

### Build

```bash
npm run build
npm run preview
```
